# Mando de campo
Vantagem de casa. Em Copa (sede majoritariamente **neutra**) é quase sempre **zero**. Entra na **V1**, mas o valor do anfitrião passa pelo **portão**.

## Fórmula
```
H_host2026 = +40 Elo [a calibrar]  para anfitrião (EUA/MEX/CAN) em solo próprio
           = 0                      nos demais (NEUTRO)
banda_mando = ±20 Elo  → entra em σ_dr nos jogos de anfitrião
```

## Por que +40 (e não +60)
Os **jogos-fantasma da COVID** ([[Jogos-fantasma COVID (mando)]]) mostram que mando é majoritariamente **viés de árbitro/torcida**; público de Copa é mais misto → efeito menor. Rebaixado de +60→+40, em banda.
